# External Projects and Repositories

This curated list contains interesting external codebases, frameworks, and projects.

*   **Scrapling** - An adaptive, high-performance Web Scraping framework for Python that handles everything from a single request to a full-scale crawl.
    [GitHub: D4Vinci/Scrapling](https://github.com/D4Vinci/Scrapling)
*   **Alex Mironov on StackBlitz** - A collection of code examples, snippets, and mini-projects.
    [StackBlitz Profile](https://stackblitz.com/@alex-mironov)
*   **RuView** - An edge AI perception system that turns commodity WiFi signals into real-time human pose estimation and presence detection without using video.
    [GitHub: ruvnet/RuView](https://github.com/ruvnet/RuView)
